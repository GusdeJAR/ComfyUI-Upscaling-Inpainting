# ComfyUI-Upscaling-Inpainting-REMBG
Este repositorio incluye varios workflow de superresolución de imágenes con baja resolución, con modelos de sobreescalado, StableDiffusion, y multimodal. En adición, viene un workflow aparte de Inpainting con el modelo de RealisticVision, segmentación de máscaras con GroundingDINO y SAMSegment, y finalmente, REMBG. Se anexa un código para métricas.
